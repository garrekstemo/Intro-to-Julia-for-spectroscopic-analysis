### Mathematical notation uncommon in other languages

```
function f(x, y)
    return \sqrt(x^2 + y^2)
end

f(3, 4)
```

compact syntax
```
g(x, y) = \sqrt(x^2 + y^2)
g(3, 4)
```

Can even write unicode characters

```
\Delta
\gamma
\pi
```


if-else
```
function h(x)
    if 0 < x < 3
        return x
    else
        return 0
    end
end

h(2)

h(5)
```

## Broadcasting

```
p(x) = x^3 - 4x^2 + 2x

A = [1 2 3;
     4 5 6;
     7 8 9]

p(A)
```

What if we want to perform this function element-wise?

```
broadcast(p, A)

p.(A)

q(x) = exp.(x) .+ x

@. q(x) = exp(x) + x
```
